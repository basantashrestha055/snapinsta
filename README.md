# Snapinsta

This is an instagram-clone, built using Next.js, Auth.js, Resend, Zustand, Tanstack Query, Shadcn UI and Socket IO

## Suggestion

Since, I have used free plan of Resend to send verification code while signing up using credentials, the email is only currently sent to me. So, I recommend signing in using your google account.

## Cloning

If you want to clone the project and run,

```bash
npm install
npm run build
npm run start
```

Remember to set up the .env file referencing to .env.example
